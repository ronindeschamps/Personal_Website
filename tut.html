About Me

My name is Ronin Deschamps, and I am a full stack software developer. Currently, I am in my third year of computer science studies at Stevens Institute of Technology. Low-level programming has been my most recent focus, learning more about Assembly language and C++. I am about to tackle a new C++ project in which I will program a modular synthesizer. I have also recently started working with Alex Wellerstein, creator of Nuke Map, on a project open source project called AWEL, Atomic Weapons Effect Library. Programming is something that I deeply enjoy as it is always challenging and engaging. I am a restless mind that needs stimulation and programming will always scratch that itch for me. When I am not programming I like to produce music and cook. I am partial to activities that allow me to create things and develop skills that make creating intuitive. I find some the greatest pleasures in surprising myself when I have made something to be proud of that is cohesive, expressive and authentic to myself.

GeoTab Shuttle Tracker

This project used the Geotab tracker device and the MyGeotabSDK library to create a live tracking system for the Maine Maritime Academy student shuttle. I wrote API calls with the MyGeotabSDK that grabbed the location data of the tracker. The data was displayed as a moving vehicle on an adaptive map I created with the Leaflet JavaScript Library. The webpage I built was integrated by another member of the apps team into the Modo Labs application that is used on the back end of the Maine Maritime website and mobile app. This project is live at https://r.mma.edu/shuttletracker/
